# Data Generation Rules

## 사용자 생성

- 총 사용자: 1,000,000
- 나이:
  - 부모: 30~55세
  - 자녀: 5~25세

## 가족 구성

- 가족 수: 250,000
- 가족 크기: 2~8명
- 역할:
  - OWNER
  - PARENT
  - CHILD

## 데이터 공유 정책

- FIFO 또는 PRIORITY
- 가족 데이터 = 인원 × 5GB

## 정책 적용 확률

| 정책 | 확률 |
|------|------|
| 시간차단 | 15~30% |
| 서비스 차단 | 20~70% |

## 데이터 선물

- 50% 가족 발생
- 부모 → 자녀
- 최대 5GB
- 일 단위 요금제(`period=DAY`)는 제외
- 1.5GB 요금제(`amount=int(1.5*GB)`)는 제외
